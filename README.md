# 🚀 SPACE SHOOTER

A simple browser-based space ship shooter game built using HTML, CSS, and JavaScript.

## 🎮 Features
- Player-controlled spaceship
- Shooting mechanics
- Enemy objects
- Simple arcade-style gameplay
- Lightweight and runs directly in browser

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📁 Project Structure
- index.html
- shooter.png
- shooter2.png
- shooter3.png
- shooter4.png
- .vscode/

## ▶️ How to Run Locally
Just open `index.html` in any browser.
