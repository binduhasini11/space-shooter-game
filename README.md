# 🚀 SPACE SHOOTER

A simple browser-based space ship shooter game built using HTML, CSS, and JavaScript.

## 🌐 Live Demo
https://binduhasini11.github.io/space-shooter-game/

## ▶️ How to Run Locally
Just open `index.html` in any browser.

## 🎮 Features
- Player-controlled spaceship
- Simple arcade-style gameplay
- Lightweight and runs directly in browser
- bgm and audio effects
- score and lives system
- Multi-avatar feature
- Power-ups :rapid mode & shield mode
- gain lives by beating the purple enemy (5 hits = kill)

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
